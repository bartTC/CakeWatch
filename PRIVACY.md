# Privacy Policy

CakeWatch does not collect, store, or transmit any personal data.

## What the app does

- Connects to the StatusCake API over HTTPS to manage your uptime monitors
- Stores API keys locally on your device
- Displays monitor data locally on your device

## What the app does NOT do

- No data is sent to any server other than the StatusCake API
- No analytics or tracking
- No telemetry
- No third-party dependencies

## Permissions explained

| Permission       | Why it's needed                          |
| ---------------- | ---------------------------------------- |
| Network access   | Communicate with the StatusCake API      |
| Local storage    | Store and retrieve API keys               |
